Sobre Nós


Este site é um projeto para a conclusão do meu curso de Técnico em Informática, da disciplina de Progrmação para a Web sob a orientação do Professor Ozias Mendes.
A inspiração de um site de receitas veio da aula sobre lista, onde o exercício proposto foi a utilizção de lista ordenadas e não ordenadas para a listagem de ingredientes e modo de preparo de uma receita bolo.
Este trabalho é consideravelmente simples dentre tudo que é possível fazer utilizando HTML e CSS. Todavia, cada linha de código é o melhor que posso ofercer, cada linha é o resultado de horas na frente do computador assistindo videoaulas, batendo cabeça com uma tag kkkk algumas vezes me rendendo ao bom e velho ctrl+c ctrl+v e muitas vezes ficando acorda até às 3h da madrugada. Agradeço ao professor Ozias pelas críticas que me levaram a tentar melhorar este trabalho.
Este projeto é o meu melhor até opresente momento!!!


Ass: Maria Vitória Carvalho
